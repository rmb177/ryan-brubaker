Morse Code Simulator

  • Dot: Click the Signal button to send a Dot.
  • Dash: Click and hold the Signal button for 1/4 of a second or more to send a Dash.
  • Word separator: Click and hold the Signal button for 1 second or more to send a word separator (a vertical bar).
  • The decoder "operates" on one character at a time. Let the communication line empty out before starting a new character. There will be a small delay before the character is output to the screen, but you can still begin the next character once the communication line is empty.
  • The decoder ignores any dot/dash combination that it doesn't understand
